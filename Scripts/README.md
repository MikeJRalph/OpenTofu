# How to run ```bash-tutorial.sh```

How to run it:
```
# 1. Save the script
nano bash-tutorial.sh    # paste the code, then Ctrl+O, Enter, Ctrl+X

# 2. Make it executable
chmod +x bash-tutorial.sh

# 3. Run it!
./bash-tutorial.sh
```

You can even pass your name and age:
```
./bash-tutorial.sh Neo 30
```
